# Калькулятор отпускных

Данный проект представляет собой микросервис на Spring Boot, предназначенный для расчёта отпускных выплат сотруднику на основе средней заработной платы за последние 12 месяцев.

Приложение реализует один HTTP API `GET /calculate` и поддерживает два варианта расчёта отпускных.

В базовом режиме пользователь передаёт сумму заработной платы за 12 месяцев и количество дней отпуска. В ответ сервис рассчитывает средний дневной заработок и возвращает сумму отпускных выплат.

В расширенном режиме пользователь может указать точные даты ухода в отпуск. В этом случае сервис выполняет расчёт отпускных с учётом выходных и праздничных дней — субботы, воскресенья и заданные праздники автоматически исключаются из оплачиваемых дней отпуска.

Средний дневной заработок рассчитывается по общеупотребимой формуле:
(сумма заработной платы за 12 месяцев / 12) / 29.3

Список праздничных дней задаётся через конфигурацию приложения и не захардкожен в бизнес-логике, что упрощает расширение и тестирование.

Проект реализован с акцентом на чистоту кода, корректное разделение ответственности и соблюдение базовых принципов проектирования. Контроллер отвечает только за обработку входных данных и валидацию запроса, сервисный слой содержит бизнес-логику расчёта, а доменная логика расчёта рабочих и оплачиваемых дней вынесена в отдельный компонент.

Для проверки корректности расчётов написаны юнит-тесты, которые тестируют бизнес-логику без использования Spring Context. Это позволяет быстро и изолированно проверять работу расчётов.

Проект выполнен в рамках тестового задания и демонстрирует базовый подход к разработке backend-приложений на Spring Boot с использованием Java.
